# LinkedList Project - The Odin Project
This project is part of [The Odin Project](https://www.theodinproject.com) curriculum.
This project is designed to introduce the linked list in JavaScript.

## Launch the project

### Prerequisites : Install Node.js
- On Windows
```powershell
winget install OpenJS.NodeJS.LTS
```
- On Linux (Debian/Ubuntu)
```bash
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### Clone the repo:
- With URL
```bash
git clone https://github.com/Louis-dub/linked_list_odin.git
```
- With SSH
```bash
git clone git@github.com:Louis-dub/linked_list_odin.git
```

### Install dependencies
```bash
npm install
```

### Launch the project
```bash
node main.js
```

### Launch tests
```bash
npm test
```

## Technologies used
- **JavaScript**: To create the Node class, the LinkedList class, and its methods
- **Jest30**: To test my functions
- **Babel7**: To translate the code so it can be understood by Jest
